## 🎉 Micords 首次上線！

感謝所有等待的貓奴們，Micords 正式開放使用。

### 新功能

- **貓咪檔案建立** — 支援照片、品種、生日、基本健康資訊
- **日常日誌** — 記錄飲食、飲水、排泄、體重
- **健康趨勢圖** — 血檢數值自動繪製成視覺化曲線圖
- **智慧提醒** — 疫苗、驅蟲、回診、用藥提醒，支援推播通知
- **醫療病歷** — 完整就診記錄，可上傳照片

### 已知限制

- 目前僅支援單隻貓咪（多貓管理預計 Q2 上線）
- 匯出功能尚在開發中
